GitHub Pages + Hexo 搭建个人博客：从零开始

发布日期: 2023年12月26日

一直想拥有一个自己的博客，记录学习点滴和分享经验。尝试过各种博客平台，但最终选择了 GitHub Pages 结合 Hexo 搭建个人博客，因为它提供了极高的自由度和完全的控制权。

以下是我搭建博客的主要步骤：

  1. 安装 Node.js 和 Git：这是 Hexo 运行的基础环境。
  2. 安装 Hexo CLI：通过 npm install -g hexo-cli 快速安装。
  3. 初始化博客：使用 hexo init myblog 创建一个新项目，然后 cd myblog && npm install 安装依赖。
  4. 配置 _config.yml：修改网站标题、作者信息、部署方式等。
  5. 选择和配置主题：Hexo 有大量精美主题可供选择，我选择了 [主题名称，例如：NexT]，并根据个人喜好进行定制。
  6. 撰写博文：使用 Markdown 格式撰写文章，非常简洁高效。
  7. 生成和部署：通过 hexo generate 生成静态文件，然后 hexo deploy 将文件推送到 GitHub Pages。
  8. 绑定自定义域名：如果需要，可以在 GitHub Pages 设置中绑定自己的域名。

搭建过程中最大的收获是深入理解了静态站点生成器的工作原理以及 Git 的版本控制。虽然初期会有一些学习曲线，但完成后，拥有一个完全属于自己的博客平台，那种满足感是无法比拟的。
